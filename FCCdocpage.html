Introduction

Important Disclaimer: THIS IS NOT an actual documentation page, just an example made for a project required to obtain a freeCodeCamp certification. For more information on the matter please refer to freeCodeCamp's homepage. And for detailed information about HTML, you can also visit W3schools.

HTML is the standard markup language for Web pages. This means that it is not a programming language. It's used to write the content of a web page and structure it with headers, images, different sections, links and a lot more.

Common tags

The way html classifies content is through the use of tags, these indicate some characteristics that the content following (or in between) such tags will have. Some of the most commonly used ones are:

  • head
  • body
  • main
  • header
  • p
  • form
  • input

Basic HTML structure

Although html documents can be written in many ways for many different purposes, we can distinguish a basic structure, like the body, containing all visible elements, and div elements separating different sections of the document.

As you can see, the language is pretty descriptive of itself, it points out the type of content to be written, so the browser, screen readers, search engines, and other tools can interpret what each part of the document is.
